 ## Sobre os projetos
Projetos que fiz para treinar minha lógica de programação

## Instruções de Instalação e Uso
### Pré-Requisitos
JDK (Java Development Kit) - Link: https://www.oracle.com/java/technologies/downloads/#java26

### Etapas
1° Instale o arquivo Principal.java do projeto que você deseja usar, você pode encontrá-lo pelo caminho "NomeDoProjeto/src/Principal.java"

2° Abra o terminal no local do arquivo Principal.java  
   Supondo que esteja na pasta Downloads  
   ```bash
   cd Downloads
   ``` 
3° Execute o código abaixo (Cria um arquivo Principal.class)  
```bash
javac Principal.java
```  
4° Agora toda vez que você digitar no terminal o código abaixo, o projeto será executado no seu terminal  
Obs: O terminal deve estar aberto no local do arquivo Principal.class.   
```bash
java Principal
```
Aviso: Caso você abrir mais de um projeto, será necessário mudar o nome dos arquivos ".class", e nesse caso, você deverá executar assim:
```bash
java NomeDoArquivo
```
